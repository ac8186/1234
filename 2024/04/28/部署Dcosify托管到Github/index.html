部署Dcosify托管到Github

2024-04-28 | 折腾 | 阅读 | 166 字 | 1 分钟

安装Docsify

在本地创建目录

cd ~/desktop
mkdir docsify
cd docsify

安装docsify-cli

npm i docsify-cli -g

初始化

docsify init ./docs

本地启动

docsify serve docs (预览/docsify/docs目录下的文件）
http://local:3000

托管到Github

创建新的仓库

任意命名为abc

git本地仓库

echo “# abc” >> README.md
git init
git add README.md
git commit -m “first commit”
git remote add origin https://github.com/xxx/abc.git
git push -u origin master

三联git更新

git add . // 添加所有文件到暂存区
git commit -m “xxx” // 提交到git仓库，xxx为提交内容的备注
git push // 推送到github

开启github的pages

进入abc仓库，settings，pages，选择运行分支，main，docs，保存
